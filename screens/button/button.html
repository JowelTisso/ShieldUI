Button

Button is an interactive element and used to give users some functionality and it is used in most of the places.

Individual usage

Button solid has two type which is button primary and button secondary.

Default :none

Container class : btn

Type class : btn-primarybtn-secondary

Button link

Button link has two types primary and secondary.

Default : none

Container class : btn-link

Type class : btn-link-primarybtn-link-secondary

Button with icon

Button can have icon on both primary and secondary button, you just have to add the class to the particular icon. Icon can be used from any source. Font awesome icon is used here.

Default : none

Icon class : btn-icon-primarybtn-icon-secondary

Floating Action Button

Floating Action Button is used for primary or common action in the screen which makes it easy for the user to interact with the app.

Default : none

Container class : btn-fab

Icon class : btn-fab-icon

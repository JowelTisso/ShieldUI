Rating

Rating is used for displaying quick overview of customers experience with the product.

Individual usage

This component is dependent on font awesome icon.You have to import font awesome icon in you project to use this component.

Rating can be used for taking or displaying user feedback.

Default :outline

Container class :rating-container

Base class : rating

Active class : rating-solid

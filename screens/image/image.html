Image

There are two types of image you will get here, responsive and rounded image.

Individual usage

This is a responsive image.You can try making the screen smaller. The image container width also has to be in percent and you can set the value accordingly.

Default :none

Image class : img-responsive

Image rounded

Rounded image, the image container width and height has to be set to equal values to get a round effect.

Default : none

Rounded class : img-rounded
